# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer and Job Match platform that helps users improve their resumes by generating ATS scores, grammar analysis, strengths, weaknesses, personalized recommendations, and resume-to-job matching using Google's Gemini AI.

---

## 🚀 Features

### 📄 Resume Management
- Upload resumes (PDF/DOCX)
- Secure cloud storage with Cloudinary
- Resume history
- View and delete uploaded resumes
- Automatic text extraction

### 🤖 AI Resume Analysis
- ATS Compatibility Score
- Grammar Score
- Professional Summary
- Resume Strengths
- Resume Weaknesses
- Missing Skills Detection
- Personalized Improvement Suggestions

### 💼 AI Job Match
- Upload Resume
- Paste Job Description
- AI Resume vs Job Comparison
- Job Match Percentage
- Matching Skills
- Missing Keywords
- Improvement Recommendations
- Previous Match History

### 🔐 Authentication
- User Registration
- Login
- JWT Authentication
- Protected API Endpoints

### ☁️ Cloud Storage
- Cloudinary Integration
- Secure Resume Upload
- Resume Deletion from Cloudinary

# Backend Setup

Create a virtual environment

```bash
python -m venv env
```  

Install dependencies

```bash
pip install -r requirements.txt
```

Run migrations

```bash
python manage.py migrate
```

Start backend

```bash
python manage.py runserver
```

---

# Frontend Setup

Go to frontend

```bash
cd frontend
```

Install packages

```bash
npm install
```

Run project

```bash
npm run dev
```
